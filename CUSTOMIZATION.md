### 🔧 Customizing SupraModerationBot
This bot is fully customizable. You can freely change the bot name, creator name, branding, embed messages, and all visible text.
Feel free to adapt everything to match your own project.

### 📝 Changing the Bot Name
Some commands and embeds may contain the name SupraModeration or SupraModerationBot.

You can replace them with any name you prefer.

Search for:

```
SupraModeration
SupraModerationBot
```
Replace with your own bot name, for example:

```
MyDiscordBot
```
### 👤 Changing the Creator Name
Some files may include references such as:

```
Created by: Sk8rfu
Developer: Sk8rfu
```
You can replace Sk8rfu with your own name, team name, or remove the line entirely.

### 📁 Where These Names Usually Appear
You will typically find these references in:

/commands/ (inside embed titles, footers, descriptions)

LogUtil.java

WarnUtil.java

Any footer text inside embeds

About or help commands (if added)

### 🔍 Quick Find & Replace
If you use VS Code:

Press Ctrl + Shift + F

Search for:

```
SupraModeration
SupraModerationBot
Sk8rfu
```
Replace them with your preferred values.

This is the fastest way to rebrand the entire bot.
